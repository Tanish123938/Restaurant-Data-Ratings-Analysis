# Restaurant-Data-Ratings-Analysis

## Project Description

The Restaurant Data & Ratings Analysis project analyzes restaurant data to understand customer ratings, performance trends, and business insights.
This project uses SQL for querying, Python for data cleaning and analysis, and Power BI for visualization, making it an end-to-end data analytics project.

## Project Objectives

- Analyze restaurant ratings and customer data
- Clean and preprocess raw data
- Perform SQL-based data exploration
- Identify trends affecting restaurant ratings
- Build interactive dashboards for insights

## Tools & Technologies

- SQL – data querying and aggregation
- Python:
  - NumPy
  - Pandas
- Jupyter Notebook
- Power BI – data visualization & dashboards

## Data Cleaning (Python)

- Performed using Pandas in Jupyter Notebook:
- Handling missing values
- Removing duplicates
- Data type corrections
- Feature engineering
- Preparing clean data for analysis and visualization

## SQL Analysis

- The SQL file includes:
- Data validation checks
- Filtering active restaurants
- Aggregation of ratings and reviews
- Group-wise analysis
- Business-related queries

## Power BI Dashboard

- The Power BI report provides:
- Rating distribution
- Top & low-rated restaurants
- Interactive filters
- Visual insights for decision-making

## Key Insights

- Rating trends vary based on restaurant features
- Clean data improves analysis accuracy
- Visualization helps identify business opportunities

## Conclusion

This project demonstrates real-world data analytics skills using SQL, Python, and Power BI.
It highlights the complete data analysis lifecycle—from raw data to business insights.
